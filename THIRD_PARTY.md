# Third-party software and models

- [rembg](https://github.com/danielgatis/rembg), MIT, provides the inference
  library and model adapters.
- The selectable segmentation weights are downloaded by rembg. Their source
  projects and license links are listed in the upstream rembg model catalogue.
  Review the selected model's terms before redistribution; this repository
  does not redistribute weight files.
- [Cog](https://github.com/replicate/cog), Apache-2.0, provides the container
  build and HTTP prediction runtime.
